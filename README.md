# Netwink

> I wrote this for computer network course.

## Build

```shell
make
sudo make install
```

## Usage

```shell
netwink [-f] [interface name] //restrict interface
        [-p] [port]//restrict port
        [-i] [IP address]// restrict IP
        [-t] [protocol name]//TCP/UDP/ICMP
        [-s] [out.txt] //save to file
        [-v] //check version
        [-h] //help
```
